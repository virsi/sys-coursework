# Черновики трёх листов формата А1

Этот файл — текстовое описание содержимого каждого листа курсовой работы. Используется для последующего рисования в MS Visio на базе шаблонов из `Материалы/*.vsdx` от преподавателя:

- `Модульная структура_01_ОБР_2025.vsdx` → лист «Стр»
- `Схема взаимодействия_01_ОБР_2025.vsdx` → лист «Схема»
- `Алгоритмы_КР_СП_ОБР_СП_2025.vsdx` → лист «Блок»

В каждом листе обязательно:

- основная надпись (штамп) в правом нижнем углу с фамилией исполнителя (Воробьев Е. А.), номером группы (ИУ5-42Б), номером варианта (5) и наименованием листа;
- рамка по ГОСТ 2.104;
- название листа сверху или в штампе.

## ЛИСТ 1 — «Модульная структура программы»

**Имя итогового файла:** `Стр_КР_СП_ИУ5-42Б_Воробьев_СП_2026.pdf`

Структура — иерархическое дерево от корня «Резидентная программа tsr.com» к трём логическим частям, далее — к процедурам и переменным каждой части.

```

                       │      (Вариант №5, ИУ5-42Б)       │

       ┌────────▼────────┐    ┌─────────▼────────┐    ┌─────────▼────────┐
```

### Подписи к блокам (что описывать в каждом)

| Блок | Описание |
|---|---|
| **Резидентная программа tsr.com** | Корневой блок. ~2,2 КБ в памяти. |
| **Сегмент данных** | Флаги функций (signaturePrintingEnabled, cursiveEnabled, translateEnabled, ignoreEnabled), буферы (cursiveSymbol, savedSymbol), адреса старых обработчиков, строки сообщений и подписи. |
| **new_int9h** | Обработчик INT 09h. Чтение порта 60h, проверка F8/F9/F1/F2, цепной вызов старого обработчика, постобработка буфера клавиатуры. |
| **new_int1Ch** | Обработчик INT 1Ch. Отсчёт 5 секунд для подписи (F8). |
| **new_int2Fh** | Обработчик INT 2Fh. AH=FFh, AL=0 — вернуть 'i'; AL=1 — выгрузка. |
| **printSignature** | Вывод трёхстрочной рамки в верх экрана. |
| **setCursive** | Подмена/восстановление шрифта буквы «З». |
| **changeFont** | Установка шрифта VGA (INT 10h, 1100h). |
| **changeFx** | Цветной индикатор F8/F9/F1/F2 в правом нижнем углу. |
| **_initTSR** | Точка входа. Очистка экрана, парсинг, проверка через INT 2Fh, либо загрузка, либо выгрузка. |
| **commandParamsParser** | Разбор PSP:80h, обработка `/?`. |
| **Установка трёх векторов** | INT 21h, функция 25xxh для INT 09h/1Ch/2Fh. |
| **int 27h** | Завершение программы с резидентным размещением. |
| **Восстановление трёх векторов** | INT 21h, 25xxh с сохранёнными значениями. |
| **INT 21h AH=49h PSP** | Освобождение PSP резидента. |
| **INT 21h AH=49h тело** | Освобождение тела резидента. |

## ЛИСТ 2 — «Схема взаимодействия с аппаратурой»

**Имя итогового файла:** `Схема_КР_СП_ИУ5-42Б_Воробьев_СП_2026.pdf`

Схема описывает связи между аппаратными компонентами компьютера, программными обработчиками резидента и областями памяти.

### Текстовая структура

```
┌─────────────┐                            ┌────────────────────┐
│ Клавиатура  │  скан-код                  │ Видеопамять B800h  │
│  (8042)     │ ─────►┐                    │  80x25, атрибуты   │
└─────────────┘       │                    └────────▲───────────┘
                      │                             │
              ┌───────▼───────┐              ┌──────┴───────┐
              │ Контроллер    │              │ INT 10h      │
              │ прерываний    │              │ (BIOS) функ. │
              │ 8259A         │              │ 02h, 13h, 11h│
              │ IRQ1 ──► INT  │              └──────▲───────┘
              │ 09h           │                     │
              └───────┬───────┘              ┌──────┴────────┐
                      │ INT 09h              │ printSignature│
              ┌───────▼─────────┐            │ setCursive    │
              │ Таблица векторов│            │ changeFx      │
              │ прерываний RAM  │            └──────▲────────┘
              │ 0000:0024h (09) │                   │ call
              │ 0000:0070h (1C) │            ┌──────┴────────┐
              │ 0000:00BCh (2F) │            │ new_int9h     │
              └───────┬─────────┘            │ new_int1Ch    │
                      │ JMP FAR              │ new_int2Fh    │
                      ▼                      └──────▲────────┘
              ┌─────────────────┐                   │
              │ new_int9h, ...  │ ─────► старый ────┘
              │ (резидент CS)   │       обработчик
              └─────────────────┘    (old_int9h*, ...)


┌─────────────┐  IRQ0      ┌──────────────────┐
│ Системный   │ ──────────►│ Контроллер 8259A │ ── INT 08h ──►
│ таймер 8253 │            └──────────────────┘  │
│  (18.2 Гц)  │                                  ▼
└─────────────┘                            ┌─────────────┐
                                           │ INT 1Ch     │
                                           │ (Timer Tick)│
                                           └─────┬───────┘
                                                 │
                                                 ▼
                                           ┌─────────────┐
                                           │ new_int1Ch  │
                                           │ (counter)   │
                                           └─────────────┘


┌─────────────┐                            ┌─────────────────┐
│ Прикладные  │ ─────► INT 2Fh ──────────► │ new_int2Fh      │
│ программы   │  AH=FFh, AL=0/1            │ (Multiplexer)   │
└─────────────┘                            └─────────────────┘
```

### Список связей

| Связь | Источник | Приёмник | Тип |
|---|---|---|---|
| Скан-код | Клавиатура | Контроллер 8259A → INT 09h | IRQ1 |
| INT 09h | Таблица векторов | `new_int9h` | far JMP |
| Старый INT 09h | `new_int9h` | (адрес из BIOS) | call dword ptr |
| Тик 18.2 Гц | Таймер 8253 | INT 08h → INT 1Ch | IRQ0 |
| INT 1Ch | Таблица векторов | `new_int1Ch` | far JMP |
| INT 2Fh | DOS / прикл. ПО | `new_int2Fh` | программное |
| Подпись на экран | `printSignature` | Видеопамять B800h через INT 10h/13h | BIOS |
| Шрифт VGA | `setCursive` → `changeFont` | Видеоконтроллер через INT 10h/1100h | BIOS |
| Индикатор | `changeFx` | Видеопамять (нижний правый угол) | через INT 10h |

## ЛИСТ 3 — «Блок-схемы алгоритмов»

**Имя итогового файла:** `Блок_КР_СП_ИУ5-42Б_Воробьев_СП_2026.pdf`

На листе размещаются блок-схемы 4 ключевых алгоритмов. Используется стандартная нотация: овал — начало/конец, прямоугольник — действие, ромб — условие, параллелограмм — ввод/вывод, шестиугольник — цикл.

## Указания по оформлению листов в Visio

1. **Шаблон рамки и штампа** — взять из любого vsdx в `Материалы/*_ОБР_2025.vsdx` (там уже есть готовая рамка по ГОСТ 2.104).
2. **Подписать каждый лист** в основной надписи (штампе): `Воробьев Е. А.`, `ИУ5-42Б`, `Вариант 5`, `2026 г.`
3. **Цвета**: использовать чёрно-белое оформление, без заливок (для чёрно-белой печати).
4. **Шрифты**: GOST type A или Arial 10-12 pt.
5. **Размер листа**: A1 (594×841 мм) либо распечатать на 4×A4 в режиме «постер».
6. **Экспорт в PDF**: File → Export → PDF, с сохранением всего листа на одной странице.

После экспорта файлы получают имена:
- `Стр_КР_СП_ИУ5-42Б_Воробьев_СП_2026.pdf`
- `Схема_КР_СП_ИУ5-42Б_Воробьев_СП_2026.pdf`
- `Блок_КР_СП_ИУ5-42Б_Воробьев_СП_2026.pdf`
