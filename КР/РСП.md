Данная резидентная программа обладает следующими возможностями: выводом на экран текстового сообщения, содержащего ФИО студента, индекс группы и номер варианта курсовой работы; модификацией русской буквы «З» на курсивный вариант; русификацией пяти латинских клавиш в кириллические буквы «ЕЖЗИК»; ограничением ввода русских букв с клавиатуры. Работа программы возможна только в текстовом режиме MS DOS.

# 2. Требования к составу и параметрам технических средств

Программа должна запускаться на компьютере со следующей минимальной конфигурацией:

1. IBM-совместимый компьютер с процессором 8086 и выше;
2. Не менее 3 Кбайт свободной оперативной памяти;
3. VGA-совместимый видеоадаптер и монитор;
4. Не менее 3 Кбайт свободного дискового пространства;
5. Стандартная PC-клавиатура.

# 3. Требования к программному обеспечению

Для работы программы необходимо наличие установленной ОС MS DOS версии 5.0 или выше. При наличии эмулятора DOSBox 0.74 и выше возможен запуск программы под операционными системами семейства Windows (Windows XP, Windows 7, Windows 10, Windows 11) и macOS.

# 4. Состав программного продукта

В состав данного программного продукта входит один файл — сама резидентная программа `tsr.com`.

# 5. Установка продукта

Для работы с программой необходимо скопировать с носителя на целевой компьютер файл `tsr.com`. Если предполагается запуск программы из ОС семейства Windows, необходимо предварительно установить эмулятор DOSBox (`http://www.dosbox.com`).

# 6. Удаление продукта

Для удаления продукта необходимо выгрузить резидентную программу из памяти (если она запущена в данный момент) повторным запуском `tsr.com` без параметров, перейти в директорию, куда был установлен продукт, и удалить файл `tsr.com`.

# 7. Характеристики системы

Исходный код, язык: Assembler.

Исполняемый код: файл `tsr.com` (2127 байт).

# 8. Подготовка к работе с программой

Подготовка к работе с программой состоит из следующих действий.

Проверка установленной операционной системы. Если установлена ОС MS DOS версии ниже чем 5.0, необходимо установить более новую версию или ОС семейства Windows с эмулятором DOSBox.

В зависимости от установленной системы алгоритм подготовки может варьироваться.

Если установлена ОС MS DOS версии 5.0 и выше: запустить ОС, установить продукт, перейти в директорию с установленным продуктом (если программа находится на диске C в папке WORK выполнить: `c:` [Enter]; `cd work` [Enter]).

Если установлена ОС Windows XP и выше: запустить ОС, установить эмулятор DOSBox, установить продукт, открыть эмулятор DOSBox, примонтировать директорию с установленным продуктом командой `mount D c:\work` [Enter], перейти на смонтированный диск командой `D:` [Enter].

# 9. Запуск программы

Запуск программы осуществляется командой `tsr.com`. Для этого нужно перейти в директорию с программой или прописать путь этой директории в переменную окружения PATH. После запуска появится сообщение «Резидент загружен!», если до запуска резидент отсутствовал в памяти, или «Резидент выгружен», если резидент уже находится в памяти и выполняется его выгрузка. В случае неверного ввода ключей будет выведено сообщение «Ошибка параметров командной строки».

# 10. Завершение работы программы

Выгрузка резидента из памяти осуществляется повторным запуском программы (`tsr.com`) без параметров. В случае успешной выгрузки в командной строке выдаётся сообщение «Резидент выгружен».
