## Параметры варианта №5

| Функция | Клавиша | Параметры |
|---|---|---|
| Вывод подписи по таймеру | **F8** | Через **5 секунд** в **верхнюю часть экрана** выводится рамка с тремя строками: «Воробьев Е. А.» / «ИУ5-42Б» / «Вариант №5» |
| Курсивный показ буквы | **F9** | Русская буква **«З»** переводится в курсивное начертание; повторное нажатие возвращает обычный шрифт |
| Русификация подмножества клавиш | **F1** | Латинские клавиши `T`, `;`, `P`, `B`, `R` (на местах русских букв `ЕЖЗИК` в раскладке ЙЦУКЕН) выводят соответственно `Е`, `Ж`, `З`, `И`, `К` |
| Ограничение ввода | **F2** | Русские буквы (строчные и прописные, cp866) блокируются — не доходят до буфера экрана; латиница, цифры и спецсимволы работают как обычно |
| Справка | `tsr /?` | Выводит описание ключей в командную строку, резидент не загружается |
| Загрузка | `tsr` | Если резидент не установлен — загружает его в память и выводит «Резидент загружен!» |
| Выгрузка | `tsr` (повторно) | Если резидент уже установлен — выгружает его и выводит «Резидент выгружен» (требование группы ИУ5-42Б) |

## Порядок сборки программы

Сборка выполняется под DOSBox (или нативный MS DOS) утилитами Turbo Assembler.

```
tasm.exe /l tsr.asm          # компиляция, артефакты tsr.obj + tsr.lst
tlink /t /x tsr.obj           # линковка в .com-формат, артефакт tsr.com + tsr.map
```

Флаги:
- `/l` — генерировать листинг (`tsr.lst`), нужен для документа ЛИТ.
- `/t` — собирать `.com` (Tiny model: код+данные в одном сегменте).
- `/x` — не создавать map-файл (можно опустить если нужен `tsr.map`).

После сборки проверить:

```
tsr.com /?                    # справка, резидент не грузится
mem.exe /P                    # текущее состояние памяти, до загрузки
tsr.com                       # выводит "Резидент загружен!"
mem.exe /P                    # видим резидент TSR ~2 КБ в списке
                              # → теперь работают F8, F9, F1, F2
tsr.com                       # повторный запуск → "Резидент выгружен"
mem.exe /P                    # резидент в списке отсутствует
```
